Find the Bot

Search for the following in the Telegram search bar and open the conversation:

@igroup_kevin_bot
2

Start the Bot

Send the following message:

/start

The Bot will greet you and prompt you to log in before you can use it.

3

Connect Your monday.com Account

Send the following message:

/login

The Bot will return an authorization link. Tap it to go to the monday.com authorization page.

📌 This step only needs to be done once. You won't need to log in again after that.
4

Authorize on monday.com

💻 Desktop
  • Click the link sent by the Bot — the monday.com authorization page will open directly
  • If not logged in, sign in with your email and password
  • Click "Authorize" or "Allow" to confirm
  • When the page shows "Authorization Successful", you're done
  • Return to Telegram — the Bot will notify you that the connection is complete
🤖 Android Chrome
  • Tap the Bot's link, enter the authorization page, then tap "Go to monday.com Authorization"
  • If an "Open in Browser" dialog appears, check "Always open links from auth.monday.com in browser" then tap Continue
  • When redirected to the download page, tap the ⋮ menu in the top-right corner → select "Open in..."
  • Enable "Desktop site" ✓
  • Refresh the page — you should now see the monday.com login form. Complete authorization.
🍎 iOS Safari
  • Tap the Bot's link, enter the authorization page, then tap "Go to monday.com Authorization"
  • When redirected to the download page, tap the ↑↓ share button at the bottom
  • Select "Open in..." → enable "Request Desktop Website" ✓
  • Refresh the page — you should now see the monday.com login form. Complete authorization.
💡 Enabling desktop mode makes monday.com treat your mobile browser as a desktop device, bypassing the mobile redirect and loading the authorization form correctly.
5

Start Using

Common Commands

/start Start the Bot
/login Connect monday.com account
/status Check current login status
/reset Clear login state (switch accounts)

FAQ

Q The authorization link shows "Cannot reach this site"?
The link may have expired. Send /login again to get a new one.
Q The Bot is not responding to my messages?
Send /status first to confirm you're logged in. If not, run /login again.
Q Can I use it on multiple devices?
Yes. As long as you use the same Telegram account, authorization only needs to be done once.
Q My mobile keeps going to the App download page — what do I do?
This is monday.com's mobile detection mechanism. Enable "Desktop site" mode in your mobile browser. For Android tap ⋮ → Open in..., for iOS tap ↑↓ → Open in.... See Step 4 for full details.
Q Is my monday.com data safe?
The Bot only uses the permissions you authorized to read/operate data. Your account password is never stored.
